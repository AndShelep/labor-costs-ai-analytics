# Відкриті дані AI Analytics

## Мета проекту

Аналіз офіційних державних статистичних даних щодо витрат підприємств на оплату праці в Україні (2005–2019) з метою виявлення регіональних відмінностей, довгострокових тенденцій та структурних змін з часом. Отримання навичок використання системи контролю версій git.

## Джерело даних

Офіційна Державна служба статистики України
Портал відкритих даних: https://data.gov.ua/dataset/d83c20f1-c2a1-4fb8-96f7-4ec9788ab41b

Набір даних: Витрати підприємств на оплату праці за регіонами України (2005–2019).

## Дослідницькі питання

1. Чи існує довгострокова тенденція зростання витрат на оплату праці протягом 2005–2019 років?

2. Витрати на утримання робочої сили значно відрізняються між регіонами. Де найбільші та найменші витрати?

3. Чи вплинули економічні кризи (2008–2009 та 2014–2015 років) на динаміку витрат на оплату праці?
