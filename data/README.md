# Дані

## Витрати підприємств на утримання робочої сили (Офіційна державна статистична інформація, що поширюється відповідно до плану державних статистичних спостережень)

Набір даних взятий з порталу https://data.gov.ua/

Посилання на набір даних: https://data.gov.ua/dataset/d83c20f1-c2a1-4fb8-96f7-4ec9788ab41b

Набір даних містить наступні поля:

- id - унікальний номер рядка спостереження
- region - область чи місто
- year - рік спостереження
- cost - витрати на утримання робочої сили
